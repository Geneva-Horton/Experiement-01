Free Time

My Elementary School Memories

One of my earlier memories of using the Internet is playing Animal Jam during class in elementary school. We would be given free time with the computers between classes, and I remember when everyone would quickly search up either Club Penguin, Cool Math Games, or, of course, Animal Jam. We would all start playing the mini games with each other and trading our items. I don't remember much from elementary school, but those computer breaks will always be a fond memory for me.
